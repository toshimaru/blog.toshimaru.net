Macを使い、今一部で流行っているらしいScalaによるWebフレームワーク、playframeworkを動かしてみます。

インストール

homebrewを使えばscalaおよびplayフレームワークは簡単にイントールできます。

brew install play
brew install scala

確認してみます。can play?

アプリケーション作成

create new play application.

play new first_app

What is the application name? 
> first_app

Which template do you want to use for this new application? 

  1 - Create a simple Scala application
  2 - Create a simple Java application
  3 - Create an empty project

> 1

OK, application first_app is created.

Have fun!

これで起動する準備は完了。

アプリケーション起動

run application.

cd first_app/
play
[first_app] $ run
[first_app] $ run

--- (Running the application from SBT, auto-reloading is enabled) ---

[info] play - Listening for HTTP on port 9000...

http://localhost:9000/ で下記の画面が確認できればOK.
