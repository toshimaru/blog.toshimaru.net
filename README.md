# blog.toshimaru.net

My blog repository powered by Jekyll.

## Create new post

    $ jekyll post YOUR_POST_TITLE

## Rake Tasks

### Initial setup

    $ rake setup

### Serve static pages

    $ jekyll serve

### Deploy to gh-pages

    $ rake deploy
