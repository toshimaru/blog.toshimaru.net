# blog.toshimaru.net

My blog repository powered by [Jekyll](https://github.com/jekyll/jekyll).

## Create new post

```console
$ bundle exec jekyll post YOUR_POST_TITLE
```

## Serve static pages

```console
$ bundle exec jekyll s
```
